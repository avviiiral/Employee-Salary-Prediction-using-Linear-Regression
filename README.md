# Salary Prediction using Linear Regression

## 📌 Project Description
This project demonstrates how **Linear Regression** can be used to predict employee salaries based on their years of experience. It is a beginner-friendly machine learning project that covers the complete workflow from data loading to model evaluation.

---

## 📂 Dataset Information
- **Dataset Name:** Employee Salary Dataset  
- **File:** `Dataset09-Employee-salary-prediction.csv`

### Columns:
- `YearsExperience` – Total years of professional experience  
- `Salary` – Annual salary of the employee

---

## 🛠️ Tools & Technologies
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## 🤖 Machine Learning Model
- **Algorithm:** Linear Regression  
- **Learning Type:** Supervised Learning  
- **Problem Type:** Regression

---

## 🔍 Project Workflow
1. Import required libraries
2. Load the dataset
3. Explore and understand the data
4. Preprocess the data
5. Split data into training and testing sets
6. Train the Linear Regression model
7. Make salary predictions
8. Visualize actual vs predicted values
9. Evaluate model performance

---

## 📊 Model Evaluation
The model is evaluated using:
- Mean Squared Error (MSE)
- R² Score

---

## ▶️ How to Run the Project
1. Open Jupyter Notebook
2. Load the notebook file `salary pred linear regration.ipynb`
3. Run all cells step by step to see predictions and visualizations

---

## 📈 Results
- The model successfully captures the linear relationship between experience and salary.
- Predictions closely match actual salary values for most cases.
