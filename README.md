# Employee Salary Prediction using Linear Regression

## 📌 Project Overview
This project focuses on predicting employee salaries based on years of experience using **Linear Regression**, one of the most fundamental machine learning algorithms.  
The goal is to understand the relationship between experience and salary and build a model that can make accurate predictions.

---

## 📂 Dataset
- **File Name:** `Dataset09-Employee-salary-prediction.csv`
- **Description:**  
  The dataset contains employee details with the following columns:
  - `YearsExperience` – Number of years of experience
  - `Salary` – Annual salary of the employee

---

## ⚙️ Technologies Used
- Python 🐍
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib / Seaborn
- Scikit-learn

---

## 📊 Machine Learning Model
- **Algorithm Used:** Linear Regression  
- **Type:** Supervised Learning (Regression)

---

## 🧪 Steps Performed
1. Import required libraries
2. Load and explore the dataset
3. Data preprocessing
4. Splitting data into training and testing sets
5. Train Linear Regression model
6. Make predictions
7. Visualize results
8. Evaluate model performance

---

## 📈 Model Evaluation
- Model performance is evaluated using:
  - Mean Squared Error (MSE)
  - R² Score
